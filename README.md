### Acceder a los elementos:

```const miArray = ["David", "Juan", "Kevin"];```

### Agregar elementos:

```const miArray = ["David", "Juan", "Kevin"];```

```console.log(miArray);```

```// Imprime: ["David", "Juan", "Kevin"];```

```miArray.push("Carlos");```

```console.log(miArray);```

```// Imprime: ["David", "Juan", "Kevin", "Carlos"];```

```miArray.unshift("Carlos");```

```console.log(miArray);```
